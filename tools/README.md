# Tools

This area owns project automation and support utilities.

- `build-scripts/` bootstrap and local workflow commands
- `content-authoring/` catalog scaffolding, previews, metrics, and diff helpers
- `content-validation/` schema validation and repository linting
- `quality/` Phase 6 budget, soak, save-health, playtest, and RC gate tooling
- `telemetry/` local event log conventions
- `migration/` save and content migration placeholders
