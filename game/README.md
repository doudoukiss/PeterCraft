# Game

This area owns runtime code, data, assets, UI, and audio placeholders.

- `code/` contains the application shell and module scaffolding.
- `data/` contains portable gameplay data, schemas, and fixtures.
- `data/content/` contains runtime-loaded shipped content catalogs for Phase 5.
- `o3de/` contains the pinned Phase 7 playable project used by the Phase 7.2 playable session loop.
- `assets/`, `audio/`, and `ui/` hold placeholder content contracts for later phases.
