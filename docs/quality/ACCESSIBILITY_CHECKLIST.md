# Accessibility Checklist

## Critical flow checks

- Core loop is playable with mouse/keyboard and controller.
- No critical instruction depends on color alone.
- Text is readable at 100%, 125%, 150%, and 180% subtitle scale.
- Subtitle background, high contrast, icon redundancy, and motion comfort are testable in the shell.
- First mission, first extraction, first repair, first creator edit, and first explain-panel open all have readable help text.

## Review severity

- `blocker`: prevents completion of a critical flow
- `serious`: flow is technically possible but confusing or fatiguing
- `minor`: polish issue that does not block understanding
